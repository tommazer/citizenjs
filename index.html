Facebook like button

class="ct like"

 Options

data-href: target url for the like. Default: current page

data-send: send button. Default: false

data-width: width of the plugin. Default: 450

data-show-faces: display facepile. Default: true
Facebook send button

class="ct send"

 Options

data-href: target url for the share. Default: current page
Facebook follow button

class="ct follow"

 Options

data-href: target url for the follower (Ex. https://www.facebook.com/zuck). Required

data-width: width of the plugin. Default: 450

data-show-faces: display facepile. Default: true
Facebook comments box

class="ct comments"

 Options

data-href: target url for the comments. Default: current page

data-width: width of the plugin. Default: 450

data-num-posts: number of posts displayed. Default: 2
Facebook like box

class="ct likebox"

 Options

data-href: target url for the like (Ex. https://www.facebook.com/vakantieveilingen.nl). Required

data-width: width of the plugin. Default: 292

data-show-faces: display facepile. Default: true

data-header: display header. Default: false

data-stream: display stream. Default: false
